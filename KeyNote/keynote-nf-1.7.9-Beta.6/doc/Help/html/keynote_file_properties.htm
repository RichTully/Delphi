KeyNote File properties

FILE PROPERTIES is a small group of settings related to the whole .KNT file, rather than particular notes inside it. For example, you can specify the format in which the file should be saved, choose whether the file should be encrypted (and specify the access passphrase requireed to open the file), add a description which will make it easier to identify the file, etc.

To change FILE PROPERTIES for the currently open file, press Alt+Enter, or select the "Properties" command from the "File" menu.

The dialog box contains four tab sheets (of which only three are visible if your KeyNote file is not encrypted). Click the links below for details about settings available on each tab.
