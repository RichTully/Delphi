Options: Tab icons

This page allows you to customize the icons used for the note tabs. You can add, insert or delete icons.

Add... button
Opens a standard Windows file selection dialog box, and adds the icon you select to the end of the list

Insert... button
Opens a standard Windows file selection dialog box, and inserts the icon you select at the current position in list.
  Warning: Inserting an icon will move all subsequent icons forward. As a result, existing notes will now display different icons.

Delete button
Deletes the selected icon from list.
  Warning: Deleting an icon will move all subsequent icons forward. As a result, existing notes will now display different icons.

Reset button
Reloads the built-in set of icons, effectively removing any customization.
